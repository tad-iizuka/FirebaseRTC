[開発用] Firebase ID Token取得ツール

これは本番クライアントの一部ではありません。curlでtoken-serverのAPIを手動テストするために ID Tokenをコピーするためだけの、ローカル専用の開発ツールです。公開ホスティングには含めないでください。

下の firebaseConfig を、Firebase Console > プロジェクトの設定 > 全般 > マイアプリ（ウェブアプリ）から取得した値で書き換えてから使ってください。
