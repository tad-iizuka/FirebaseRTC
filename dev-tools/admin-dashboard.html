PTT 管理者ダッシュボード(開発者用)

これは本番クライアントの一部ではありません。/admin/rooms API を手動で確認するための ローカル専用の開発ツールです。公開ホスティングには含めないでください。 閲覧には adminUsers/{uid}.permissions に rooms:monitor が 付与されているFirebaseアカウントでのサインインが必要です (dev-tools/grant-admin-permission.js で付与)。
注意: このファイルを file:// で直接開くと Firebase Authが auth/unauthorized-domain エラーを返しサインインできません。 cd dev-tools && python3 -m http.server 8000 等でローカルサーバーを立て、 http://localhost:8000/admin-dashboard.html として開いてください。
